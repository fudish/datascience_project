# datascience_project

Il dataset "access_electricity.csv" contiene dati sull'accesso alla elettricità della popolazione in vari paesi del mondo, suddivisi per anno dal 2012 al 2016. Ogni riga rappresenta un paese con informazioni quali nome del paese, codice paese, nome e codice della serie di dati, e percentuale della popolazione con accesso alla elettricità per ciascun anno. Inoltre, il dataset include la classificazione continentale di ogni paese, il che permette di analizzare le differenze regionali in termini di accesso all'elettricità.

L'analisi eseguita sul dataset si concentra sull'andamento dell'accesso alla elettricità nel periodo considerato, evidenziando trend di crescita o stagnazione per continente e singoli paesi. Si possono osservare significative differenze tra continenti, con alcune aree come l'Europa e il Nord America che raggiungono facilmente il 100% di copertura, mentre altre regioni come l'Africa mostrano valori molto più bassi e in crescita meno rapida. Questa analisi permette di mettere in luce le disparità nello sviluppo energetico mondiale e può servire come base per studi più approfonditi o interventi mirati.
